== Description ==

With WP Geo-Tags you can add the meta-tags geo.region, geo.placename, geo.position and ICBM to every post and page separately. Support: [Linkaufbau](http://www.linkarbeit.net/)

== Installation ==

1. Extract `wp-geo-tags.1.0.zip`.
2. Upload the `wp-geo-tags`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
